# PopOS Setup

My personal Pop!_OS development environment backup.

## Includes

- GNOME settings
- Keyd keyboard layout
- Sublime Text config
- Shell setup
- Package installer

---

## Setup

```bash
git clone https://github.com/sandeshgorde/popos-setup.git
cd popos-setup
bash scripts/setup.sh
```

---

## Restore Existing System

```bash
bash scripts/install.sh
bash scripts/setup.sh
```
